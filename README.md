# Hijri Calendar 2026

A responsive Islamic calendar web application displaying Hijri dates with corresponding Gregorian dates for February 2026 to December 2026.

## Features

- Displays 11 monthly calendars from Ramadan 1447 to Rajab 1448 (starting February 2026)
- Accurate Hijri-Gregorian date conversion following specified date ranges
- Beautiful responsive design with CSS Grid, works from 270px and above
- Islamic holidays highlighted with special markers
- Hijri dates displayed in bold, Gregorian dates in normal weight
- Weekend days (Saturday and Sunday) have light golden background
- Each month card includes designer credit: "Designed By: Azmat Ali"
- Print-friendly styles
- Mobile-first responsive design

## Design Highlights

- **Color Scheme**: Deep teal/emerald (#0d9488) for headers and warm amber/gold (#f59e0b) for highlights
- **Typography**: Modern sans-serif fonts with Poppins for headings
- **Layout**: CSS Grid for calendar structure and responsive columns
- **Visual Effects**: Subtle gradients, shadows, and hover animations
- **Accessibility**: Proper contrast ratios and focus states

## Technical Implementation

- **HTML**: Semantic markup with proper heading hierarchy
- **CSS**: Custom properties, flexbox, grid, and responsive breakpoints
- **JavaScript**: Accurate Hijri-Gregorian conversion algorithm
- **Responsive**: Mobile (1 column), tablet (2 columns), desktop (3 columns)

## Files

- `index.html`: Main HTML structure
- `styles.css`: Styling and responsive design
- `script.js`: Date conversion and calendar rendering logic
- `script.ts`: Original TypeScript source (for reference)

## Usage

Simply open `index.html` in any modern web browser to view the calendar.

## Browser Compatibility

Works in all modern browsers (Chrome, Firefox, Safari, Edge).
